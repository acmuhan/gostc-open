# 商业化功能任务清单

## P0 — 核心基础（必须先完成）

- [ ] P0-1 余额 / 积分账户体系：充值、消费、退款、赠送、资金流水
- [ ] P0-2 订单系统：购买、续费、充值统一走订单，支持幂等和过期
- [ ] P0-3 套餐计费统一化：四类隧道统一套餐、价格、周期、流量、节点限制

## P1 — 支付 & 权益

- [ ] P1-4 易支付对接：支持支付宝、微信、QQ钱包等充值和支付
- [ ] P1-5 CDK 功能：支持兑换余额、积分、套餐、用户组
- [ ] P1-6 礼包码 / 活动码：支持多人领取、次数限制、活动奖励
- [ ] P1-7 用户组体系：控制额度、权限、节点、套餐、隧道数量
- [ ] P1-8 用户组 + 套餐灵活付费：支持免费、仅积分、仅余额、二选一支付

## P2 — 运营管控

- [ ] P2-9 隧道自动续费：到期前自动扣余额或积分续费
- [ ] P2-10 流量包 / 月流量重置：支持月流量限制、额外流量包、用尽策略
- [ ] P2-11 封禁用户 / 隧道：支持封禁用户、封禁隧道、解封和日志
- [ ] P2-12 节点负载展示：展示在线状态、隧道数、连接数、速率、流量
- [ ] P2-13 操作日志 / 审计日志：记录管理员操作、支付回调、配置修改等

## P3 — 体验优化

- [ ] P3-14 自定义 404 页面：支持默认或节点 / 域名级 404 页面
- [ ] P3-15 链接配置 / 菜单配置：支持隐藏注册、菜单、按钮、页脚链接等
- [ ] P3-16 邮件模板：支持验证码、支付成功、到期提醒、封禁通知等模板
- [ ] P3-17 限制邮箱域：支持邮箱域白名单 / 黑名单

## P4 — 增长 & 合规

- [ ] P4-18 优惠券系统：支持满减券、折扣券、新用户券、指定套餐可用
- [ ] P4-19 邀请返利 / 分销：支持邀请码、注册奖励、充值返佣
- [ ] P4-20 实名 / 备案辅助：支持实名状态、域名备案信息、管理员审核
- [ ] P4-21 工单 / 客服系统：支持用户提交工单、管理员回复、状态流转
